# screen-time
iOS application to track screen time and stop you being on your phone


## Ideas of features to implement:
 - main app that stores all your tracking data and can see graphs and progress
 - Goal setting in main app
 - Notifications when you've been on your phone for a long time (can set what you want this to be). Tells you your time in this session as well as how long total
 - Optional push notifications for friends as well so they can shame you if you've been on your phone for a while
 - Optional blocker - some amount of seconds you must wait before opening app (for certain apps)
 - ability to sort apps into your own buckets/categories
 - Notification when you unlock your phone asking if you're going to be productive or entertain
 - Sync with computer - can track time there as well
 - With multi-device sync - If you have one device open you can't open a different one
 - Icons of apps tell you how long you've been on it that day/week
 - Setting to set start day/time zone
 - Location blocking
 - Places where you can not adjust settings
 - Can only adjust settings at home?
